Kako dobiti token:
1. Otvori GitHub - Settings - Developer settings - Personal access tokens - Tokens (classic)
2. Klikni "Generate new token (classic)"
3. Naziv: "PROVIDENTIA Blog"
4. Expiration: "No expiration"
5. Selektiraj samo "repo" (all)
6. Klikni "Generate token" i kopiraj cijeli kod
Token ce biti spreman u lokalnom storage - trebat ce ti samo prvi put.
